 Power BI — Hotel Reservation Analysis
This project analyzes hotel booking patterns, cancellations, guest preferences, and special requests using interactive Power BI dashboards.

Key Insights
- Weekday bookings are significantly higher than weekend bookings.
- Cancellations increase during peak months.
- Majority of guests request amenities like breakfast, high floor, and late checkout.
